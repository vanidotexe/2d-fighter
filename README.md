# 2D Fighter

[![Engine](https://img.shields.io/badge/Godot-4.x-000000?labelColor=000000&logo=godotengine&logoColor=ffffff)](https://godotengine.org)
[![License](https://img.shields.io/badge/License-MIT-000000?labelColor=000000&logo=github)](LICENSE)

A tiny educational 2D fighter built in Godot to learn core systems: mechanics, states, HUD, AI, and project structure.

## Features

- Character state machine: IDLE, WALK, CROUCH, JUMP, ATTACK, HURT
- Attack routing and hitbox playback
- Basic projectile
- Health updates
- Round timer and end-of-round results
- Pause menu handling
- Simple AI with distance-based decisions, timed actions, and walking/attacking
